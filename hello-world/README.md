# Hello World — Light / Dark / Party

A simple hello world web app with three visual modes: **Light**, **Dark**, and **Party**.

## Modes

- **Light** — Light gray background, dark text, minimal and clean
- **Dark** — Dark background, light text, easy on the eyes
- **Party** — Animated gradient background, bouncy text, and playful button effects

## How to Run

No build step or dependencies required. Open `index.html` in a web browser:

```bash
# From the hello-world directory
open index.html

# Or use a simple HTTP server (optional)
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## Files

- `index.html` — Main HTML structure
- `styles.css` — Styles for all three modes
- `app.js` — Mode switching logic

## Usage

Click the **Light**, **Dark**, or **Party** buttons to switch between modes. The active mode is highlighted.
